MPS060602¶

Python wrapper of MPS-060602 Acquisition Card.

MPS-060602 is a Dual Channel, 16 Bit High Speed IEPE Vibration ICP Acquisition Card, with USB Support and LabVIEW Software. Documentation.

WARNING: This package is a wrapper of its DLL, only working on Windows.

Manufacturer¶

Beijing MPS electronic, official website (Chinese).
